# Stremo Product Roadmap

Our vision is to build the most creator-centric live streaming platform in the world.

## Phase 1: Foundation (Current)
- [x] Real-time streaming (WebRTC/RTMP)
- [x] VOD Archiving & Playback
- [x] Basic Creator Profiles
- [x] Real-time Chat & Stats
- [x] Screen Sharing

## Phase 2: Professionalization (Q3 2024)
- [ ] **Adaptive Bitrate Streaming (ABR)**: Implement HLS/DASH for smooth playback across all network conditions.
- [ ] **Advanced Transcoding**: Use GCP Transcoder API for multi-resolution support (144p to 4K).
- [ ] **Monetization**: Integrated tipping, subscriptions, and ad-insertion.
- [ ] **Mobile Apps**: Native iOS and Android applications for on-the-go streaming.

## Phase 3: Ecosystem & AI (Q4 2024)
- [ ] **AI Highlights**: Automatically generate short clips from long streams using Gemini AI.
- [ ] **Collaborative Streaming**: Multi-guest streams with low-latency audio/video mixing.
- [ ] **Creator Analytics**: Deep insights into viewer behavior and retention.
- [ ] **Global Expansion**: Deploy PoP (Point of Presence) servers in emerging markets.

## Phase 4: Scale (2025+)
- [ ] **Decentralized Delivery**: Peer-to-peer assisted CDN to reduce costs and latency.
- [ ] **VR/AR Streaming**: Support for immersive 360-degree live experiences.
- [ ] **Open API**: Allow third-party developers to build on top of the Stremo protocol.
